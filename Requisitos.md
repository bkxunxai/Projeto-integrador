## DOCUMENTO DE REQUISITOS.


## Requisitos Funcionais:


##### RF01 - Login:

***O sistema terá a opção de registrar sua conta para salvar suas informações.***

##### RF02 - Gestão Financeira:

***O sistema deve calcular o lucro final subtraindo custos de insumos e manutenção da receita da colheita.***

##### RF03 - Mapa de Produtividade:

***O usuário deve conseguir marcar quais áreas da fazenda renderam mais sacas.***

##### RF04 - Histórico de Pragas:

***O sistema deve manter um historico de infestações passadas para consulta.***

##### RF05 - Registro de Aplicações:

***O sistema deve permitir registrar qual defensivo foi usado e em qual data.***

##### RF06 - Tipos de cultura:

***O sistema deve permitir registrar os tipos de cultura.***

##### RF07 - Indicações:

***O sistema terá opção para escolher qual cultura terá mais prioridade para serem fornecidas indicações mais frequentes.***

##### RF08 - Visualização de preços:

***O sistema terá uma aba destinada à visualização de preços dos produtos das empresas da região registrada no sistema.***

##### RF09 - Diagnóstico:

***O sistema terá a função de permitir o envio de imagens para obter um diagnóstico da planta.***

##### RF10: Visualizar Clima:

***O sistema terá uma aba para a visualização da previsão do tempo em tempo real e futura.***

## Requisitos Não-Funcionais:


##### RNF01 - Usabilidade:
***A interface deve ter botões grandes e letras legíveis para facilitar o uso de pessoas com dificuldades em aparelhos digitais.***

##### RNF02 - Disponibilidade Offline:

***O sistema deve permitir o registro de dados sem internet, sincronizando quando houver sinal*** 

##### RNF03 - Desempenho:
***O cálculo de lucro deve ser processado em menos de 2 segundos.***

##### RNF04 - Banco de Dados atualizado:

***O banco de dados será atualizado frequentemente, assim para garantir os dados mais precisos.***

##### RNF05 - Banco de Dados Culturas:

***Após ter registrado os tipos de cultura, será feito o download automaticamente do banco de dados sobre essa cultura para ser disponibilizado offline, assim para poder ter consulta quando necessário, independentemente do local.***

##### RNF06 - Notificações:

***O sistema enviará notificações frequentemente para alertar como está o manejo.***

##### RNF07 - Clima:

***O sistema enviará notificações sobre condições climáticas.***

##### RNF08 - Coleta de informações:

***O sistema coleta informações do usuário, assim para se adequar ao estilo dele.***
