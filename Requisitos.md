## DOCUMENTO DE REQUISITOS.


### Requisitos Funcionais:
RF01 - Registro de Aplicações: O sistema deve permitir registrar qual defensivo foi usado e em qual data (Baseado na Pergunta 8).

RF02 - Gestão Financeira: O sistema deve calcular o lucro final subtraindo custos de insumos e manutenção da receita da colheita (Baseado na Pergunta 10).

RF03 - Mapa de Produtividade: O usuário deve conseguir marcar quais áreas da fazenda renderam mais sacas (Baseado na Pergunta 14).

RF04 - Histórico de Pragas: O sistema deve manter um log de infestações passadas para consulta (Baseado na Pergunta 15).

### Requisitos Não-Funcionais:
RNF01 - Usabilidade: A interface deve ter botões grandes e letras legíveis para facilitar o uso no campo (Adequação à idade e ocupação).

RNF02 - Disponibilidade (Offline): O sistema deve permitir o registro de dados sem internet, sincronizando quando houver sinal 

RNF03 - Desempenho: O cálculo de lucro deve ser processado em menos de 2 segundos.
